# Eden_Editor:_Toolbar

Source: https://community.bohemia.net/wiki/Eden_Editor:_Toolbar

---

The Toolbar is displayed at the top of the Workspace. It provided quick access to commonly used actions and tools.

### Toolbar Buttons

Icon

Description

| 

| Open a dialog to create a new scenario

| 

| Open a dialog to choose a scenario to open

| 

| Save the current scenario. If it has not been saved yet, open a dialog to save the scenario with specific name.

| 

| Open Workshop Scenario Management window, from which you can publish the scenario to the Steam Workshop. Only scenario which is already saved can be published.

| 

| Undo the last completed action.

| 

| Redo the last undone action.

| 

| Toggle the transformation widget off.

| 

| Toggle the translation widget.

| 

| Toggle the rotation widget.

| 

| Toggle the area scaling widget.

| 

| Toggle the Area Widget.

| 

| Toggle between global and local coordinate space for the transformation widget.
- Global - widget is rotated in cardinal directions (X axis points east, Y axis points north, and Z axis points up).

- Local - widget is rotated together with the selected entity (X axis points right, Y axis points forward, and Z points above the entity).

| 

| Toggle between translation parallel to the surface level and parallel to the sea level.
- Surface Level - preserve entity altitude above surface (e.g., terrain, building floor, etc.)

- Sea Level - preserve entity altitude above sea level.

| 

| Toggle ground snapping. When on, entity will be snapped to the ground surface when it gets near it during translation editing.

| 

| Toggle or set the translation grid.

| 

| Toggle or set the rotation grid.

| 

| Toggle or set the area scaling grid.

| 

| Open scenario attributes dialog.

| 

| Toggle between Scene (3D) and Map (2D) view.

| 

| Toggle flashlight. At night, it will light up the immediate surroundings of the camera.

| 

| Toggle vision mode.
- Normal

- Night Vision

- Thermal Vision

| 

| Select the scenario phase.

| 

| Open tutorial dialog. You can pick from various courses which will guide you through basic Editor features.
